#  Приложение Movies
## Тестовое задание для компании Polydev
Приложение работающее с REST API c сервисом [TMDB][https://www.themoviedb.org/].

**Язык**: Swift.
**Фреймворк**: UIKit, верстка кодом без Storyboard.
**Архитектура**: VIPER.
**Используемые библиотеки**: KingFisher.
